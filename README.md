# Starblast
Hello! This is an attempt at an old simple space shooter implementation! Written in Lua using the lovely Love2D library.

## Libraries:
This game uses a collection of existing code and libraries designed for Love2D to handle some complex things.
- [Hump](https://github.com/vrld/hump) - Camera, Finite State Machine, Vector Math.
- [Evolved](https://github.com/BlackMATov/evolved.lua) - Entity Component System (ECS).

## Assets:
This game uses an asset pack by [Gustavo Vituri](https://gvituri.itch.io/space-shooter).
